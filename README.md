## How to run?

<br>

Requires Node.js v4+ to run.

Clone this repository or download and unzip this zip file.

Open it in your IDE.

1. Install npm:

```
$ npm install
```


2. To run just type:

```
$ npm start
```


For full compatibilty please use Chrome.
